# GenAI Contract Analysis & Risk Assessment Bot

This project is a GenAI-powered legal assistant for Indian SMEs.

## Features
- Contract summary
- Clause-by-clause explanation
- Risk scoring (Low/Medium/High)
- Unfavorable clause detection
- Safer suggestions
- Multilingual support (English + Hindi - planned)

## Impact
Helps SMEs understand contracts, reduce legal risks, and negotiate better.
